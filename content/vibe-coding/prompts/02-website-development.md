# Prompt 模板：網站開發

## 1. 網站設計生成 Prompt

```
請幫我設計一個 [網站類型] 的網頁，使用 HTML + Tailwind CSS。

## 視覺風格
- 整體風格：[簡約 / 活潑 / 專業 / 文青]
- 主色：[色碼或顏色名]
- 輔色：[色碼或顏色名]
- 參考網站：[URL]（我喜歡它的 [具體元素]）

## 頁面區塊（由上到下）
1. [區塊名稱]：[描述]
2. [區塊名稱]：[描述]
3. [區塊名稱]：[描述]

## 要求
- 手機版自適應（RWD）
- 使用 Tailwind CSS CDN
- 圖片先用 placeholder
- 中文內容
```

## 2. 頁面風格延伸 Prompt（Cursor 用）

```
參考 @index.html 的設計風格（顏色、字體、間距、元件樣式），
幫我建立 [頁面名稱].html。

頁面內容：
1. [區塊描述]
2. [區塊描述]
3. [區塊描述]

要求：
- header 和 footer 與 index.html 完全一致
- 使用相同的 CSS class 命名慣例
- 保持視覺一致性
```

## 3. RWD 修正 Prompt

```
我的網站在手機上顯示有問題：
[描述問題，例如：「文字超出螢幕」「圖片太大」「兩欄變成重疊」]

請幫我修正，確保在以下裝置上都正常顯示：
- 手機（375px 寬）
- 平板（768px 寬）
- 桌面（1280px 寬）

使用 Tailwind CSS 的 responsive prefix（sm:, md:, lg:）。
```

## 4. 導覽列與漢堡選單 Prompt

```
請幫我建立一個響應式導覽列：

桌面版：
- 左邊：LOGO（文字）
- 右邊：[選單項目1] [選單項目2] [選單項目3] [CTA 按鈕]

手機版：
- 左邊：LOGO
- 右邊：漢堡選單圖示
- 點擊後展開全螢幕選單

要求：
- 使用 Tailwind CSS
- JavaScript 控制開合
- 平滑的動畫效果
- 滾動時固定在頂部（sticky）
```

## 5. 表單設計 Prompt

```
請幫我建立一個 [聯絡 / 報名 / 訂閱] 表單：

欄位：
- [欄位名稱]（[類型：文字/Email/下拉選單/多行文字]，[必填/選填]）
- [欄位名稱]（[類型]，[必填/選填]）

要求：
- 使用 Tailwind CSS 美化
- 包含前端驗證（必填檢查、Email 格式）
- 送出按鈕有 hover 效果
- 送出後顯示成功訊息
- 手機版友善
```

## 6. 動畫效果 Prompt

```
請幫我為網站加入以下動畫效果：

1. Hero 區塊：標題和副標題淡入 + 從下方滑入
2. 卡片區塊：滾動到該區塊時，卡片依序淡入
3. CTA 按鈕：hover 時有微微放大效果
4. 頁面切換：平滑滾動到指定區塊

使用 CSS animation + Intersection Observer API。
不要使用任何外部動畫庫。
效果要優雅微妙，不要太花俏。
```
