# Prompt 模板：Landing Page

## 1. Landing Page 文案規劃 Prompt

```
我要做一個 Landing Page，目的是 [收集 Email / 銷售產品 / 報名活動]。

## 背景資訊
- 產品/服務：[描述]
- 目標受眾：[誰]
- 他們的痛點：[列出 2-3 個]
- 我的解決方案：[描述]
- 獨特賣點：[你跟別人不一樣的地方]

請幫我撰寫以下區塊的文案：

1. **Hero**：一句話標題（10 字以內）+ 副標題（30 字以內）+ CTA 按鈕文字
2. **Pain Points**：3 個痛點，每個用一句話描述
3. **Solution**：解決方案描述（50 字以內）
4. **Benefits**：4 個好處，每個用一句話 + emoji
5. **Social Proof**：設計 2 個推薦語（可虛擬，後續替換）
6. **CTA**：最終行動呼籲文案
7. **FAQ**：5 個常見問題與回答

語調：[專業但親切 / 輕鬆幽默 / 緊迫有力]
語言：繁體中文
```

## 2. Landing Page 開發 Prompt

```
請根據以下文案，用 HTML + Tailwind CSS 建立一個高轉換率 Landing Page。

[貼上文案]

## 設計要求
- 單頁式，無導覽列
- Hero 區塊佔滿螢幕高度，深色背景
- CTA 按鈕在頁面中出現至少 3 次
- 使用者永遠不需要滾動超過一個螢幕就能看到 CTA
- 手機版優先設計
- 載入速度要快（不用外部圖片庫）

## 表單設計
表單欄位：[姓名 + Email / 只有 Email]
表單送出方式：先用 `action="#"` placeholder
送出後顯示感謝訊息

## 進階效果
- 滾動時各區塊淡入
- CTA 按鈕有微動畫（pulse 效果）
- 倒數計時器（可選）
- 已有 [N]+ 人 [下載/報名]（社群證明數字）
```

## 3. A/B 測試文案 Prompt

```
我有一個 Landing Page，目前的標題是：
「[現有標題]」

轉換率不太理想。請幫我產出 5 個替代標題方案，每個方案使用不同的文案策略：

1. 數字驅動（用具體數字吸引注意）
2. 痛點導向（直接點出問題）
3. 好奇心引發（讓人想了解更多）
4. 社群證明（暗示很多人已經在用）
5. 直接利益（告訴讀者他會得到什麼）

每個方案給出：標題 + 副標題 + 簡短說明為什麼這樣寫。
```
