# Prompt 模板：除錯與修正

## 1. 萬用除錯 Prompt

```
我遇到了一個問題，請幫我修正。

## 錯誤訊息
[貼上完整的錯誤訊息]

## 期望行為
[描述你期望發生什麼]

## 實際行為
[描述現在實際發生什麼]

## 相關程式碼
[貼上相關的程式碼片段，或在 Cursor 中用 @file 引用]

請一步步分析原因，然後提供修正方案。
```

## 2. 視覺修正 Prompt

```
我的網頁有以下視覺問題需要修正：

[描述問題，或貼上截圖]

具體要修改的地方：
1. [元素] — 現在是 [現狀]，我想要 [期望]
2. [元素] — 現在是 [現狀]，我想要 [期望]
3. [元素] — 現在是 [現狀]，我想要 [期望]

請只修改這些問題，不要動其他部分。
```

## 3. 手機版修正 Prompt

```
我的網站在桌面版看起來正常，但手機版有以下問題：

裝置：[iPhone / Android / 平板]
螢幕寬度：[375px / 768px / 其他]

問題清單：
1. [描述問題]
2. [描述問題]
3. [描述問題]

請用 Tailwind CSS 的 responsive classes 修正。
只修改有問題的部分，保持桌面版不變。
```

## 4. 功能不正常 Prompt

```
我的 [功能名稱] 不正常工作。

## 操作步驟
1. [我做了什麼]
2. [然後做了什麼]
3. [期望看到什麼]
4. [實際看到什麼]

## 環境
- 瀏覽器：[Chrome / Safari / Firefox]
- 裝置：[桌面 / 手機]

## 已嘗試
- [我已經試過什麼]

請幫我找出問題並修正。
```

## 5. 程式碼重構 Prompt

```
以下程式碼可以正常運作，但我覺得它可以更好。

[貼上程式碼]

請幫我：
1. 找出可以改善的地方
2. 重構讓程式碼更容易理解
3. 確保重構後功能完全不變
4. 解釋你改了什麼以及為什麼

保持使用相同的技術（HTML/CSS/JS），不要引入新的框架。
```
