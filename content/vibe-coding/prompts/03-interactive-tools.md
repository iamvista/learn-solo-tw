# Prompt 模板：互動工具開發

## 1. 計算機工具 Prompt

```
請幫我建立一個 [類型] 計算機，使用 HTML + Tailwind CSS + JavaScript。

## 功能說明
計算目的：[描述]
目標使用者：[誰會用]

## 輸入欄位
1. [欄位名稱] — [類型：數字/下拉選單/滑桿]（範圍：[最小值]-[最大值]）
2. [欄位名稱] — [類型]（選項：[列出選項]）
3. [欄位名稱] — [類型]

## 計算邏輯
[描述計算公式，不需要寫程式碼，用自然語言說明]

## 結果展示
- 主要數字：[顯示什麼]
- 輔助資訊：[額外說明]
- 視覺化：[進度條/圖表/色彩反饋]

## 視覺要求
- 深色主題，圓角卡片
- 輸入即時計算（不需要按按鈕）
- 結果區塊要醒目
- 手機版友善
```

## 2. 診斷測驗 Prompt

```
請幫我建立一個「[測驗名稱]」互動測驗，使用 HTML + Tailwind CSS + JavaScript。

## 測驗設計
- 題目數量：[N] 題
- 題目類型：單選題
- 結果類型：[N] 種（[列出結果名稱]）

## 結果定義
| 結果 | 描述 | 適合什麼 | 建議 |
|------|------|----------|------|
| [A]  | ...  | ...      | ...  |
| [B]  | ...  | ...      | ...  |
| [C]  | ...  | ...      | ...  |

## 題目內容
請根據以上結果定義，設計 [N] 道題目。
每題 3-4 個選項，每個選項對應不同的結果類型加分。

## 視覺要求
- 一次顯示一題，有進度條
- 選擇後自動跳下一題
- 結果頁面要有大標題、描述、建議
- 加入「分享結果」和「再測一次」按鈕
- 配色活潑但專業
```

## 3. 互動工具美化 Prompt

```
我已經有一個可運作的 [工具名稱]，但視覺很陽春。
請幫我美化，加入以下效果：

1. 載入動畫：計算時顯示 loading spinner
2. 數字動畫：結果數字從 0 遞增到最終值
3. 色彩反饋：根據結果好壞顯示不同顏色（綠=好，黃=普通，紅=需改善）
4. 微互動：按鈕 hover 效果、輸入框 focus 效果
5. 社群分享：一鍵複製結果文字

保持現有功能不變，只做視覺升級。
```
