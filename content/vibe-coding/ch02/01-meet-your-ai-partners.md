# 2-1 認識你的 AI 夥伴：Claude、ChatGPT、Cursor

## 學習目標

- 認識三大主流 AI 工具的定位與差異
- 瞭解什麼場景用什麼工具最有效
- 註冊並準備好所有帳號

## 內容大綱

### 1. AI 工具全景圖（3 分鐘）
- 對話型 AI：Claude、ChatGPT、Gemini
- 開發型 AI：Cursor、Windsurf、Bolt
- 設計型 AI：v0、Galileo AI
- 本課程聚焦：Claude + Cursor（最強組合）

### 2. Claude — 你的思考夥伴（5 分鐘）
- Anthropic 出品，擅長長文理解與邏輯推理
- 三種使用方式：
  - claude.ai（網頁對話）
  - Claude App（桌面/手機）
  - API（進階整合）
- 免費版 vs. Pro 版差異
- **實際示範**：用 Claude 做專案規劃

### 3. ChatGPT — 萬用瑞士刀（3 分鐘）
- OpenAI 出品，生態系最完整
- 適合：快速問答、搜尋替代、圖片生成
- 免費版 vs. Plus 版差異
- 與 Claude 的互補使用策略

### 4. Cursor — 你的 AI 編輯器（7 分鐘）
- 基於 VS Code 的 AI-native 編輯器
- 為什麼不用 VS Code + Copilot？
- Cursor 的三大殺手功能：
  - **Tab**：自動補全（寫一半它幫你寫完）
  - **⌘+K**：行內編輯（選取 → 描述 → 修改）
  - **Agent Mode**：整個功能一次做完
- 免費額度說明與使用建議

### 5. 帳號註冊實作（5 分鐘）
- 手把手帶你註冊 Claude 帳號
- 手把手帶你註冊 Cursor 帳號
- 登入後的初始設定建議
