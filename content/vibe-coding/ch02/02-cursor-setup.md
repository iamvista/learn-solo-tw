# 2-2 Cursor 安裝與基本操作

## 學習目標

- 完成 Cursor 安裝與初始設定
- 熟悉基本介面與操作
- 建立第一個專案資料夾

## 內容大綱

### 1. 下載與安裝（3 分鐘）
- 到 cursor.com 下載
- Mac / Windows 安裝流程（螢幕錄製）
- 首次啟動的設定選項說明

### 2. 介面導覽（5 分鐘）
- 左側欄：檔案總管、搜尋、原始碼管理
- 編輯區：分割視窗、標籤頁
- 終端機（Terminal）：你的命令列工具
- 右側：AI Chat 面板
- 常用快捷鍵一覽表

### 3. 重要設定（5 分鐘）
- 語言設定：繁體中文介面
- AI Model 選擇：Claude Sonnet（推薦）
- Auto-save 開啟
- 字體大小調整
- 推薦安裝的擴充功能：
  - Chinese Language Pack
  - Prettier（程式碼格式化）
  - Live Server（即時預覽）

### 4. 建立第一個專案（5 分鐘）
- 建立資料夾：`my-first-project`
- 用 Cursor 開啟資料夾
- 建立 `index.html` 檔案
- 用 AI Chat 請它幫你寫一個簡單的自我介紹頁面
- 用 Live Server 在瀏覽器中預覽

### 5. 三種 AI 互動方式快速體驗（5 分鐘）
- **Chat（⌘+L）**：問問題、請它寫程式
- **Inline Edit（⌘+K）**：選取文字，請它修改
- **Tab 補全**：開始打字，讓它幫你完成
- 什麼時候用哪一種？（決策樹）
