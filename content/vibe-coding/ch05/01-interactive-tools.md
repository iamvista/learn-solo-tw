# 5-1 打造互動工具：計算機 / 診斷測驗

## 學習目標

- 用 AI 打造有互動性的網頁工具
- 學會處理使用者輸入和動態顯示
- 完成一個可以實際使用的小工具

## 內容大綱

### 1. 互動工具的商業價值（3 分鐘）
- 互動工具 = 免費的行銷利器
- 案例：房貸試算器、BMI 計算機、性格測驗
- 為什麼這些工具能帶來流量和信任？
- 你可以為自己的領域做一個

### 2. 專案一：專業計算機（12 分鐘）
- 選一個適合你領域的計算機
  - 自由工作者：報價計算機
  - 健身教練：每日熱量計算機
  - 行銷人：ROI 計算機
- 用 AI 規劃功能：
  - 輸入欄位設計
  - 計算邏輯
  - 結果展示方式
- 在 Cursor 中開發（Agent Mode）
- 加入漂亮的 UI（圓角卡片、色彩反饋、動畫）

### 3. 專案二：診斷測驗（12 分鐘）
- 什麼是診斷測驗？（5-10 題選擇題 → 個人化結果）
- 案例：「你適合哪種 AI 工具？」診斷測驗
- 設計流程：
  - 定義 3-4 種結果類型
  - 設計 5-8 道題目
  - 每個選項配分邏輯
- 用 AI 一次做完：題目設計 + 程式碼 + 視覺
- 結果頁面加上分享按鈕

### 4. 讓工具更專業（5 分鐘）
- 加入載入動畫（計算中…的效果）
- 結果可以複製 / 截圖分享
- 加入「再測一次」按鈕
- 收集使用者 email（進階，Ch5-3 教）

### 5. 部署你的工具（3 分鐘）
- 跟第四章一樣 push 到 GitHub + Vercel
- 一個 repo 可以有多個專案嗎？
- 建議：每個工具一個 repo

- [ ] 完成一個診斷測驗，部署上線
