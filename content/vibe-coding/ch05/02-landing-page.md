# 5-2 建立名單蒐集頁 (Landing Page)

## 學習目標

- 理解 Landing Page 的行銷邏輯
- 用 AI 打造高轉換率的名單蒐集頁
- 串接表單收集使用者 Email

## 內容大綱

### 1. Landing Page 是什麼？為什麼重要？（5 分鐘）
- Landing Page = 單一目標的頁面（不是官網首頁）
- 目標：讓訪客做一件事（留下 email、購買、報名）
- 為什麼你需要 Landing Page？
  - 收集潛在客戶名單
  - 驗證市場需求
  - 行銷活動的著陸頁
- 與一般網站的差異：無導覽列、單一 CTA、高聚焦

### 2. 高轉換 Landing Page 的結構（8 分鐘）
- **Hero**：一句話說明價值主張 + 視覺
- **Pain Points**：你遇到的問題是…
- **Solution**：我們的解決方案
- **Benefits**：你會得到什麼（3-5 個好處）
- **Social Proof**：見證、數據、logo 牆
- **CTA**：行動呼籲（表單 / 按鈕）
- **FAQ**：消除疑慮
- 每個區塊的文案撰寫 Prompt

### 3. 實作：用 AI 做 Landing Page（12 分鐘）
- 場景：「免費 Vibe Coding 入門指南」電子書 Landing Page
- Step 1：跟 Claude 規劃文案（使用文案模板 Prompt）
- Step 2：在 Cursor 中生成頁面
- Step 3：加入表單（先用靜態 HTML form）
- Step 4：視覺調整與行動版優化
- Step 5：加入簡單的動畫效果

### 4. 表單收集方案（5 分鐘）
- 方案一：Google Forms（最簡單）
- 方案二：Tally.so（免費、好看）
- 方案三：Formspree（嵌入式表單）
- 實作：用 Tally.so 建立表單並嵌入頁面
- 收到資料後怎麼管理？匯出 CSV

### 5. 提升轉換率的技巧（3 分鐘）
- 倒數計時（製造緊迫感）
- 社群證明（已有 500+ 人下載）
- 彈出式表單（頁面滾動觸發）
- A/B 測試的概念（先有流量再說）
