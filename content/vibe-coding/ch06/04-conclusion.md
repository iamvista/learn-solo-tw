# 6-4 結語：從使用者進化為創造者

## 學習目標

- 回顧整個課程的學習旅程
- 建立「創造者心態」
- 展望你的下一步

## 內容大綱

### 1. 回顧你的旅程（5 分鐘）
- 第一章：你連 Vibe Coding 是什麼都不知道
- 第二章：你裝好了工具，跟 AI 說了第一句話
- 第三章：你學會了 Prompt 的藝術
- 第四章：你做出了一個完整的網站，全世界都看得到
- 第五章：你做了互動工具、Landing Page，串接了外部服務
- 第六章：你知道怎麼變現、怎麼持續成長
- **你用了大約 20 小時，從零到有了一項全新的能力**

### 2. 創造者心態（5 分鐘）
- 世界上有兩種人：消費者和創造者
- 消費者：看到問題 → 找現成工具 → 將就用
- 創造者：看到問題 → 想到解法 → 動手做出來
- 你已經是創造者了
- 「但我還是需要 AI 幫忙啊？」
  - 畫家也需要畫筆，攝影師也需要相機
  - 工具不會減損你的創造力，它放大你的創造力

### 3. 真實案例分享（5 分鐘）
- 3-5 位學員 / 社群成員的真實故事
- 他們用 Vibe Coding 做了什麼？
- 對他們的生活 / 事業帶來什麼改變？
- （預留位置，課程上線後補充真實案例）
