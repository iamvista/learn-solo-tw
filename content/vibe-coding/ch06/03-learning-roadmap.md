# 6-3 持續學習的路線圖

## 學習目標

- 認識 Vibe Coding 之後的進階學習方向
- 建立持續學習的習慣和資源清單
- 根據你的目標選擇學習路線

## 內容大綱

### 1. 你已經走了多遠？（3 分鐘）
- 課程開始時：零基礎
- 現在你會：
  - 跟 AI 有效溝通
  - 規劃和設計網站
  - 開發和部署完整網站
  - 串接外部服務
  - 基本的 SEO 和變現
- 你的能力在市場上的位置（能力光譜圖）

### 2. 三條學習路線（12 分鐘）

#### 路線 A：全端開發者（技術深化）
- 下一步學什麼？
  - React / Next.js（框架）
  - Tailwind CSS（深入）
  - Node.js + Express（後端基礎）
  - PostgreSQL / MongoDB（資料庫）
- 推薦資源和學習順序
- 時程估計：3-6 個月到可接案水準
- 適合：想往技術方向發展的人

#### 路線 B：產品打造者（商業應用）
- 下一步做什麼？
  - 打造自己的 SaaS 產品
  - 學習付費系統串接（Stripe）
  - 使用者驗證（Auth）
  - 資料分析與優化
- 推薦的 No-Code / Low-Code 工具
- 時程估計：1-3 個月做出 MVP
- 適合：想創業或做副業的人

#### 路線 C：AI 應用專家（AI 深化）
- 下一步學什麼？
  - AI API 串接（OpenAI、Claude API）
  - 建立 AI-powered 的工具
  - Prompt Engineering 進階
  - AI Agent 開發
- 推薦資源和社群
- 時程估計：持續學習，技術日新月異
- 適合：對 AI 特別有興趣的人

### 3. 學習資源精選（5 分鐘）
- **免費**：
  - YouTube 頻道推薦（中英文各 3 個）
  - freeCodeCamp
  - MDN Web Docs
- **付費**（值得投資）：
  - Cursor Pro（如果你常用）
  - Claude Pro（更大的使用量）
  - 進階線上課程推薦
- **社群**：
  - Vibe Coding 臺灣（2,400+ 人）
  - 各技術的 Discord / Slack 社群

### 4. 建立學習習慣（5 分鐘）
- 每天 30 分鐘 > 週末 5 小時
- 「做中學」比「看教學」有效 10 倍
- 每週做一個小專案
- 記錄你的學習過程（部落格 / 社群貼文）
- 找到學習夥伴或導師

## 課後行動

- [ ] 選擇你的學習路線（A/B/C）
- [ ] 設定每週的學習時間塊
- [ ] 加入至少一個技術社群
- [ ] 規劃你的下一個專案
