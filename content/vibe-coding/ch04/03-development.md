# 4-3 開發：一步步打造完整網站

## 學習目標

- 用 Cursor Agent Mode 完成完整網站開發
- 學會專案檔案結構管理
- 處理開發過程中的常見問題

## 內容大綱

### 1. 專案結構（5 分鐘）
- 一個網站專案長什麼樣？
  ```
  my-website/
  ├── index.html       ← 首頁
  ├── about.html       ← 關於我
  ├── services.html    ← 服務項目
  ├── contact.html     ← 聯絡方式
  ├── css/
  │   └── style.css    ← 自訂樣式
  ├── js/
  │   └── main.js      ← 互動功能
  └── images/          ← 圖片
  ```
- 為什麼要分檔案？（好管理、好修改）
- 在 Cursor 中建立這個結構

### 2. 用 Agent Mode 開發首頁（12 分鐘）
- 把上一堂課的設計成果交給 Agent
- Prompt：「請根據現有的 index.html 設計，補充以下功能：1. 導覽列的手機版漢堡選單 2. 平滑滾動效果 3. 回到頂部按鈕」
- 看 Agent 自動：
  - 修改 HTML 結構
  - 新增 JavaScript 程式碼
  - 測試功能是否正常
- 你的角色：驗證結果、提出修正

### 3. 完成所有頁面（10 分鐘）
- 一頁一頁做，每頁確認後再進下一頁
- 頁面間的共用元素（導覽列、頁腳）保持一致
- Prompt 技巧：
  - 「所有頁面的 header 和 footer 都跟 index.html 一樣」
  - 用 @file 引用共用元件

### 4. 常見開發問題處理（5 分鐘）
- 圖片不顯示？路徑問題的排查
- 手機版跑版？RWD 斷點調整
- 頁面間風格不一致？統一 CSS 變數
- 載入速度太慢？圖片壓縮 + Lazy Loading

### 5. 最終檢查清單（3 分鐘）
- [ ] 所有頁面的連結都能正常跳轉
- [ ] 手機版 / 平板 / 桌面版都正常顯示
- [ ] 圖片都能正確載入
- [ ] 沒有明顯的文字錯誤
- [ ] 頁面載入速度可以接受

## 課後作業（必做）

- [ ] 完成所有頁面的開發
- [ ] 跑一遍最終檢查清單
- [ ] 截圖 3 個你最滿意的頁面，分享到社群
