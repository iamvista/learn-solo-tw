# 4-4 部署上線：讓全世界看到你的作品

## 學習目標

- 學會使用 GitHub 管理程式碼
- 用 Vercel 一鍵部署網站
- 設定自訂網域（選用）

## 內容大綱

### 1. 什麼是「部署」？（2 分鐘）
- 你的網站目前只在你的電腦上
- 部署 = 放到網路上讓所有人都能看到
- 免費部署方案：Vercel（本課程使用）

### 2. GitHub 基礎（8 分鐘）
- GitHub 是什麼？你的「程式碼雲端硬碟」
- 註冊 GitHub 帳號
- 在 Cursor 中連結 GitHub
- 三個重要概念（簡化版）：
  - **Repository（倉庫）**：你的專案資料夾
  - **Commit（提交）**：存檔一個版本
  - **Push（推送）**：上傳到 GitHub
- 實作：把你的網站推上 GitHub
  - 在 Cursor Terminal 中操作
  - 完整指令示範（不用背，AI 會教你）

### 3. Vercel 部署（7 分鐘）
- 註冊 Vercel 帳號（用 GitHub 登入）
- 匯入 GitHub Repository
- 點一下 Deploy → 完成！
- 你的網站現在有一個 `xxx.vercel.app` 網址了
- 每次你更新 GitHub，Vercel 自動重新部署

### 4. 自訂網域（5 分鐘，選用）
- 為什麼要自訂網域？（專業感 + 品牌）
- 從哪裡買網域？（Namecheap、GoDaddy、Gandi）
- 價格：大約 NT$300-500/年
- 在 Vercel 設定自訂網域（DNS 設定教學）
- SSL 自動設定（你的網站會有 🔒）

### 5. 部署後的檢查（3 分鐘）
- 用手機打開你的網站
- 分享網址給朋友看看
- Google PageSpeed Insights 測速
- 社群媒體分享預覽（OG Image）

## 恭喜！你的第一個網站上線了！ 🎉
