# 4-1 企劃：把你的想法變成具體規格

## 學習目標

- 學會從零開始規劃一個網站專案
- 產出一份完整的需求規格文件
- 為下一堂課的開發做好準備

## 內容大綱

### 1. 本章目標（2 分鐘）
- 接下來四堂課，你會從零做出一個完整網站
- 專案選擇：「個人品牌官網」或「產品銷售頁」（二選一）
- 為什麼選這兩個？因為最實用、最容易變現

### 2. 專案選擇與定位（5 分鐘）
- 個人品牌官網適合誰？自由工作者、講師、顧問
- 產品銷售頁適合誰？創業者、副業經營者
- 帶學員做選擇：用 3 個問題判斷你適合哪個
- Vista 的選擇示範：以個人品牌官網為例

### 3. 用 AI 做專案規劃（10 分鐘）
- Step 1：跟 Claude 描述你的目標受眾和需求
- Step 2：請 Claude 幫你規劃頁面結構
  - 首頁、關於我、作品集、服務項目、聯絡方式
- Step 3：每個頁面的內容區塊規劃
- Step 4：視覺風格定調
  - 色彩：暖色系 / 冷色系 / 中性色
  - 風格：專業簡約 / 活潑創意 / 文青質感
- 完整對話示範

### 4. 撰寫需求規格文件（8 分鐘）
- 用課程提供的模板填寫
- 必填項目：
  - 專案名稱與一句話描述
  - 目標受眾
  - 頁面結構（sitemap）
  - 每頁的內容區塊
  - 視覺風格參考（至少 2 個參考網站）
  - 技術選擇：HTML/CSS/JS（本課程）
- 選填項目：
  - 特殊功能需求
  - SEO 關鍵字
  - 行動呼籲（CTA）策略

### 5. 素材準備清單（3 分鐘）
- 文字內容：自我介紹、服務說明
- 圖片：個人照片、作品截圖
- 沒有素材怎麼辦？AI 幫你生成（文案 + placeholder 圖片）

## 課後作業（必做）

- [ ] 完成需求規格文件（用課程模板）
- [ ] 找到 2 個你喜歡的參考網站，截圖保存
- [ ] 準備好你的文字素材（至少自我介紹 + 3 項服務/作品描述）
