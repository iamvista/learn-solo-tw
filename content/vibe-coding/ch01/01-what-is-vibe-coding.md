# 1-1 什麼是 Vibe Coding？為什麼你該學

> 🎯 免費試看

## 學習目標

- 理解 Vibe Coding 的定義與核心理念
- 認識 AI 輔助開發的現況與趨勢
- 建立「非工程師也能做產品」的信心

## 內容大綱

### 1. 開場：一個真實案例（3 分鐘）
- Vista 如何用一個下午做出一個完整的線上工具
- 過程中「一行程式碼都沒手動寫」
- 這不是未來——這是現在

### 2. Vibe Coding 是什麼？（5 分鐘）
- **定義**：用自然語言描述需求，讓 AI 幫你寫程式碼
- 與傳統程式開發的差異（表格對比）
  - 傳統：學語法 → 寫程式 → 除錯 → 部署
  - Vibe Coding：描述需求 → AI 生成 → 驗證調整 → 部署
- 名稱由來：Andrej Karpathy 的推文
- 不是「不用學」，而是「學的東西不一樣了」

### 3. 為什麼現在是最好的時機？（5 分鐘）
- AI 模型能力的指數級成長（GPT-3 → GPT-4 → Claude 3.5 → 現在）
- 開發工具的成熟（Cursor、Bolt、v0、Replit Agent）
- 部署門檻歸零（Vercel、Netlify 一鍵部署）
- 真實數據：非工程師用 AI 做出的產品案例

### 4. 誰適合學 Vibe Coding？（3 分鐘）
- 有想法但不會寫程式的創業者
- 想提升效率的行銷人 / PM
- 想做 side project 的自由工作者
- 對「寫程式」感到畏懼但好奇的人

### 5. 這門課會帶你到哪裡？（2 分鐘）
- 課程路線圖預覽
- 你的第一個作品：課程結束時你會做出什麼
