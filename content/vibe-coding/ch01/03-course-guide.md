# 1-3 課程導覽與學習建議

## 學習目標

- 瞭解課程結構與學習路徑
- 設定合理的學習期望
- 加入社群、準備好學習環境

## 內容大綱

### 1. 課程地圖（3 分鐘）
- 六個章節的邏輯關係
  - Ch1-2：觀念建立 + 工具準備
  - Ch3：Prompt 核心技能
  - Ch4：第一個完整專案
  - Ch5：進階應用
  - Ch6：商業化 + 持續成長
- 建議學習節奏：每週 2-3 堂，4 週完成

### 2. 學習建議（5 分鐘）
- **邊看邊做**：每堂課都跟著操作，不要只看影片
- **不要怕犯錯**：AI 最大的優勢是你可以無限重來
- **善用社群**：卡關了？先問 AI，再問社群
- **記錄過程**：螢幕錄影 or 截圖你的成果

### 3. 需要準備什麼？（3 分鐘）
- 一臺電腦（Mac / Windows / Linux 都行）
- 穩定的網路連線
- 以下帳號（下一章會手把手帶你註冊）：
  - Claude（免費版即可開始）
  - Cursor（免費版每月有額度）
  - GitHub（免費）
  - Vercel（免費）

### 4. 社群與資源（2 分鐘）
